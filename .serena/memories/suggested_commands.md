# Suggested Commands

- Install dependencies: `pnpm install`
- Run dev server: `pnpm dev`
- Build for production: `pnpm build`
- Preview production build: `pnpm preview`
- Astro CLI: `pnpm astro ...`
- Useful Darwin shell commands: `pwd`, `ls`, `find`, `rg`, `sed`, `git status --short`

No test, lint, or format scripts are currently defined in `package.json`; use `pnpm build` as the narrow verification command for substantive site changes.
